Curriculum Vitae
================

Romain Brault's CV.

How to
~~~~~~

To obtain the pdf file ::

    git clone https://github.com/RomainBrault/Curriculum-Vitae
    cd Curriculum-Vitae
    chmod +x compile
    ./compile -f

The resulting file is in the subfolder bin/ and called RomainBraultCV.pdf.


Dependencies
~~~~~~~~~~~~

Compilation requires:

- TexLive (2016)
